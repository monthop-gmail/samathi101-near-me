# Samathi101 Near Me (สาขาสถาบันพลังจิตตานุภาพใกล้ฉัน)

แอปพลิเคชันเว็บ (PWA) ออกแบบมาสำหรับการค้นหา "สาขาสถาบันพลังจิตตานุภาพ" ที่ใกล้เคียงพิกัดปัจจุบันของคุณที่สุด และช่วยอำนวยความสะดวกในการค้นหาสาขาทั่วประเทศได้อย่างรวดเร็วผ่านระบบแผนที่

🌐 **Live Demo:** [samathi101-near-me.pages.dev](https://samathi101-near-me.pages.dev/)

---

## 🎯 ฟีเจอร์หลัก (Features)

1. **ระบบระบุตำแหน่ง (Geolocation Focus)**
   - ค้นหาสาขาที่อยู่ใกล้ตัวที่สุด 5 อันดับแรกโดยอัตโนมัติ (Locate Me)
   - เปิดนำทางผ่าน Google Maps ได้ทันทีจากป๊อปอัปแผนที่
   
2. **ระบบแผนที่แบบโต้ตอบ (Interactive Map)**
   - แสดงจุดปักหมุดสาขาทั่วประเทศ พร้อมซูมไปยังจุดที่เกี่ยวข้องเมื่อทำการค้นหา
   - ข้อมูลรายละเอียดของแต่ละสาขา เช่น เวลาทำการ ผู้ดูแล เบอร์โทรศัพท์ ต้นสังกัดจังหวัดและภาค

3. **แผงค้นหาและแสดงผลอัจฉริยะ (Adaptive UI)**
   - UI แบบ Bottom Sheet ที่ออกแบบมาเพื่อมือถือ (Mobile-First) โดยเฉพาะ 
   - ระบบจะแบ่งพื้นที่หน้าจอ (แผนที่ 25% / ข้อมูล 75%) เวลาค้นหา เพื่อให้เห็นภาพรวมของแผนที่อยู่เสมอ
   - ค้นหาข้อมูลครอบคลุมทั้ง: ชื่อสาขา, รหัสสาขา, หรือพื้นที่จังหวัด/ภูมิภาค

4. **เทคโนโลยี PWA (Progressive Web App)**
   - สามารถกด "Add to Home Screen" ติดตั้งลงบนหน้าจอมือถือ/แท็บเล็ตได้ราวกับแอปพลิเคชันพื้นเมือง (Native App)
   - **Network-First Service Worker**: ระบบจัดการการดึงข้อมูลที่รับประกันว่าผู้ใช้จะได้โค้ดเวอร์ชันล่าสุดเสมอตอนที่ออนไลน์ แต่ยังสามารถเปิดใช้งานแอปแบบออฟไลน์ได้เวลาที่ขาดอินเทอร์เน็ต

5. **ระบบจัดการแคช (Cache Management)**
   - ควบคุมอายุข้อมูลด้วยไฟล์ `_headers` ของ Cloudflare เพื่อป้องกันอดีตไฟล์แคชค้างในตัวเบราว์เซอร์

---

## 🛠 เทคโนโลยีที่ใช้ (Tech Stack)

- **Frontend:** Vanilla HTML5, CSS3, JavaScript (ES6+)
- **Map Engine:** [Leaflet.js](https://leafletjs.com/) (OpenStreetMap API)
- **Deployment & Hosting:** Cloudflare Pages (ผสานการทำงานกับ GitHub Actions)
- **Data Tools:** มีสคริปต์ `fetch_and_process.py` (Python) เพื่อให้สามารถดึงข้อมูลใหม่จากฐานข้อมูลศูนย์และปรับโครงสร้างการแมปปิ้งภูมิภาคเองได้ (เผื่อกรณีต้องการอัปเดตล่วงหน้า)

## 🚀 การรันโปรเจกต์ (Local Development)

การรันโปรเจกต์นี้บนเครื่องคอมพิวเตอร์แบบออฟไลน์ ไม่จำเป็นต้องติดตั้ง Node.js หรือ build-tools ใดๆ เพียงแค่มี Local HTTP Server เช่น:

**หากมี Python:**
```bash
python -m http.server 8000
```
แล้วเปิดเบราว์เซอร์ไปที่ `http://localhost:8000`

**หากใช้ VS Code:**
แนะนำให้ติดตั้ง Extension อย่าง **Live Server** และทำเพียงคลิกเปิดจากไฟล์ `index.html` ก็จะทำงานได้ทันที

---

## ⚙️ ระบบ CI/CD (Continuous Deployment)

โปรเจกต์นี้ผูกระบบอัตโนมัติเข้ากับ **GitHub Actions** (ไฟล์ `.github/workflows/deploy.yml`)
ทุกครั้งที่มีการอัปเดตโค้ดลงบน branch `main` ระบบจะทำการ Deploy โค้ดใหม่ขึ้นสู่บัญชี **Cloudflare Pages** โดยทันที

**สิ่งสำคัญเมื่อนำไปใช้เอง (Environment Secrets):**
ต้องมีการตั้งค่าใน Repository Secrets ของ GitHub ดังนี้:
- `CF_API_TOKEN`: Cloudflare Global/API Token ที่มีสิทธิ์สั่งอัปเดตระบบ Pages
- `CF_ACCOUNT_ID`: Cloudflare Account ID
